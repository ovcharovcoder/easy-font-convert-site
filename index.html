Questions?

Simple answers, no fluff

Does it work with OTF?

Yes – both TTF and OTF are fully supported.

Can I convert back from WOFF to TTF?

No, this extension is one‑way (TTF/OTF → WOFF/WOFF2). That’s the most common web workflow.

What’s the difference between WOFF and WOFF2?

WOFF2 uses Brotli compression and gives 30‑50% smaller files. WOFF has slightly better browser support (IE9+). The extension lets you pick both.

Is it really free?

Yes. Open source (MIT). No hidden costs, no telemetry, no ads.

Where can I report issues or request features?

GitHub issues — I actively maintain the project.

Do I need to restart VS Code after install?

No, the extension works immediately. You can also reload window if you prefer.
